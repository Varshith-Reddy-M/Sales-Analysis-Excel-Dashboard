# Sales Analysis Dashboard using Excel

## Objective
To analyze sales data and identify key revenue drivers using Excel.

## Tools Used
- Microsoft Excel
- Power Query
- Pivot Tables
- Excel Charts and Slicers

## KPIs
- Net Revenue
- Gross Revenue
- Average Order Value (AOV)
- Return Rate
- Average Discount
  
## Analysis Performed
- Revenue by Region
- Revenue by Product
- Revenue by Time (Year,Quarter)
- Return Analysis
- Average delivery time by Region

## Key Insights
- Identified top-performing products
- Observed customer purchasing behavior using AOV
- Analyzed return patterns across regions
- Compared revenue for coupon vs non-coupon orders

## Conclusion
This project demonstrates end-to-end data cleaning, analysis, and dashboard creation using Excel.
